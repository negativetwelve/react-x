## Getting Started

Install `@react-x/touchable` using `yarn`:

```shell
yarn add @react-x/touchable
```

## Usage

```javascript
import {
  TouchableHighlight,
  TouchableOpacity,
  TouchableWithoutFeedback,
} from '@react-x/touchable';
```
